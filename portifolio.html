Dashboard de vendas

Projeto : PETGOV BI

Cliente : Universidade de Pernambuco

Linguagens : Excel, PowerBI, R

Preview : Não publicado

A primeira etapa foi uma seleção entre os alunos da instituição para seguirem como ingressantes do projeto, parte da seleção consistia na elaboração de dashboard com dados ficticios de uma pizzaria para avaliar visão de negócio, pensamento analítico entre outros fatores de cada aluno.
